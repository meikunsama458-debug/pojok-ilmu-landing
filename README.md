# Pojok Ilmu Registration Website

A responsive registration website for Pojok Ilmu, an English & Public Speaking tutoring platform.

## Features
- Responsive landing page
- Dark mode toggle
- Tutor carousel
- Program and pricing section
- Registration form
- Supabase backend integration
- WhatsApp CTA
- Netlify deployment

## Tech Stack
- React
- Vite
- Tailwind CSS
- Supabase
- Netlify
